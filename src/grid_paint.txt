格子の塗り分け
m行n列のマス目に対して、白と黒の石を埋めていくとします。 
いずれの石も上下左右のどこかで異なる色の石と接するような配置方法は
何通りあるかを計算するプログラムを作成してください。

入力ルール
引数として２つの整数値がm, nの順で渡されます。
m, nは 0 < (n, m) < 10程度です。
処理ルール
与えられた行、列(m * n)の格子を白と黒の２色の石で埋めた場合、
いずれの石も上下左右のいずれかのマスで異なる色の石と接するような配置パターン数を計算してください。
白と黒の石を全て入れ替えた場合に石の配置が一致するパターンはそれぞれ別の配置とみなします。

仕様に反する入力を受け取った場合、終了コード1でプログラムを終了してください。

出力ルール
パターン数を標準出力に出力してください。
サンプル
$ ./YOUR_APP 2 2
6
白の石をx, 黒の石をoを用いて表すと、2行2列の格子で条件を満たす配置パターンは以下のようになります。

1.
o o
x x

2.
o x
o x

3.
o x
x o

4.
x x
o o

5.
x o
x o

6.
x o
o x
1行1列の格子では、上下左右に別の色の石を置くことが出来ないため答えは0となります。

$ ./YOUR_APP 1 1
0
以下のケースは仕様に反する入力なので、出力をせずに終了コード1でプログラムを終了します。

// 引数なし
$ ./YOUR_APP

// 不正な文字列
$ ./YOUR_APP abc
